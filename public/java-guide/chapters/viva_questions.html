Module 1: Fundamentals

Q: What is the main difference between Procedural and OOP?
A: Procedural programming focuses on functions and logic, while OOP focuses on objects that bundle data and behavior together.
Q: What is a Constructor?
A: A special method used to initialize objects. It is called when an object of a class is created.

Module 2: Encapsulation & Interfaces

Q: Why do we use Encapsulation?
A: To hide the internal state of an object and require all interaction to be performed through an object's methods (data hiding).
Q: Can an interface have fields?
A: Yes, but they are implicitly public, static, and final (constants).

Module 3: Polymorphism

Q: What is the difference between Overloading and Overriding?
A: Overloading occurs within the same class (same name, different params). Overriding occurs between parent and child classes (same name, same params).
Q: Can you override a static method?
A: No, static methods are bound to the class, not the instance. They can be hidden, but not overridden.

Module 4: Relationships

Q: Difference between Aggregation and Composition?
A: Aggregation is a weak association (child can exist independently). Composition is a strong association (child cannot exist without parent).

Module 5-7: Design Patterns

Q: When would you use the Singleton pattern?
A: When you need exactly one instance of a class, accessible globally (e.g., Database connection, Logger).
Q: What is the Factory pattern?
A: A creational pattern that provides an interface for creating objects in a superclass, but allows subclasses to alter the type of objects that will be created.

Module 8-10: JavaFX & UML

Q: What is the Stage in JavaFX?
A: The Stage is the top-level container (the window) of a JavaFX application.
Q: What does a Sequence Diagram show?
A: It shows object interactions arranged in time sequence.
